## Git Command

### restore
- `git restore <modified file>` : 파일을 수정하기 전 상태로 되돌린다.
- `git restore --staged <file>` : 스테이징된 파일을 언스테이징한다.
- `git restore --source=<hash> <file>` : 특정 커밋의 파일을 수정하기 전 상태로 되돌린다.

### fetch
- `git fetch` : 원격 저장소의 변경사항을 로컬 저장소로 가져온다.
